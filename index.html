Manual de Operación Interactivo

Este manual interactivo proporciona a los operadores las instrucciones para operar de forma segura y eficiente el controlador de dosificación industrial. El objetivo es estandarizar los procedimientos de operación, configuración básica y respuesta a alarmas a través de una experiencia práctica.

Este controlador industrial está diseñado para automatizar y supervisar procesos de dosificación, llenado y carga de líquidos. Permite preseleccionar una cantidad de producto y controla una válvula y una bomba para dosificar esa cantidad exacta, garantizando precisión y seguridad.

La Seguridad es lo Primero

⚠️

¡ADVERTENCIA!

La operación incorrecta del equipo puede resultar en daños materiales o lesiones. Siga siempre los procedimientos de seguridad.

Interruptor de Emergencia (OFF)

Simulador de Operación Estándar

Siga las instrucciones para completar una dosificación:

Paso 1: Preparación para un Nuevo Lote.

Presione la tecla [RESET] para borrar los datos del lote anterior.

Paso 2: Ingresar la Cantidad.

Use el teclado numérico para ingresar 1500 kg.

Paso 3: Confirmar la Preselección.

Presione la tecla [Set] para confirmar.

Paso 4: Iniciar la Dosificación.

Presione la tecla [Start] para comenzar.

Paso 5: Proceso en curso.

La dosificación se completará automáticamente. Puede pausar con [Stop].

Guía de Alarmas y Mensajes de Error

Mensaje en Pantalla Significado Acción Requerida
ERR OFF active Se presionó el interruptor de paro de emergencia OFF. 1. Asegúrese de que la condición de emergencia esté resuelta.
2. Gire y libere el interruptor OFF.
3. Presione [Reset] para borrar la alarma.
ERR MIN FLOW El caudal es más bajo de lo esperado. 1. El proceso se detiene.
2. Verifique si hay obstrucciones, si la bomba funciona o si el tanque de suministro está vacío.
3. Solucione la causa y presione [Reset].
ERR S.-Break NK1 Falla en el sensor de flujo (rotura de cable). 1. El proceso se detiene.
2. Notifique inmediatamente a Mantenimiento.
3. No intente continuar hasta que sea reparado.
ERR.NO GROUNDING La conexión a tierra no está detectada. 1. Verifique que la pinza de puesta a tierra esté conectada correctamente.
2. Una vez corregido, presione [Reset].
Pres. too less / great La cantidad preseleccionada está fuera de los límites. 1. Presione [Reset].
2. Ingrese una cantidad válida dentro del rango permitido.
